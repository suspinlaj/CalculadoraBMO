# 🤖 **Calculadora BMO**

**Calculadora BMO** es una aplicación funcional con un diseño único inspirado en el entrañable personaje BMO de la serie *Hora de Aventuras*. El objetivo de este proyecto es demostrar que hasta la herramienta más básica puede tener personalidad y un diseño visual atractivo estilo *pixel art*.

## 📘 **Acerca de este proyecto**

Este proyecto fue desarrollado como un ejercicio de clase para poner en práctica la lógica de programación y el manejo de interfaces en Android (utilizando **Kotlin**). En lugar de conformarme con una calculadora estándar y genérica, decidí aplicar mi pasión por la ilustración y el diseño visual para crear una experiencia divertida, utilizando una paleta de colores pastel, tipografía retro y formas redondeadas.

## 🔧 **Funcionalidades**

- **Operaciones Matemáticas Básicas**: Soporte para sumas (+), restas (-), multiplicaciones (x) y divisiones (/).
- **Funciones Adicionales**: Cálculo de porcentajes (%), uso de decimales (.) y botón de cambio de signo positivo/negativo (+/-).
- **Gestión de Entradas**: Botón de borrado total (C) para limpiar la pantalla y botón de retroceso para corregir errores rápidamente.
- **Interfaz Dinámica**: Un visor principal que imita la cara sonriente de BMO y un "bocadillo" de diálogo estilo *pixel art* en la parte superior donde se muestran los números ingresados y los resultados de las operaciones.

## ⚙️ **Estado del proyecto**

Finalizado. Se trata de un proyecto educativo de clase con sus funcionalidades básicas operativas y su diseño UI/UX completado. Probado y ajustado para emuladores de tamaño *Medium Phone*.
